Notes

The mark is a top-down stage; the clay dot is downstage-centre — the performer's mark (the ★ used on riders). Typeface is Jost, light weight, generous tracking, lowercase. Keep clear space around the lockup equal to the height of the square. Minimum wordmark width ≈ 120px; below that, use the mark alone. For print or offline use, outline the text in a vector editor so it doesn't depend on the web font.
